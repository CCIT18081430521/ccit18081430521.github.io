#关于搭建psoc4.2框架时出现的NET Framework 3.5错误的解决方法

##第一步
先找到自己电脑的控制面，找到程序与功能里面的，启动/关闭window功能选项。
里面的第一个功能就是NET Framework 3.5功能。
直接启用电脑会进行子那个下载。
如若下载没有问题 解决了
如若有问难请参考第二步

##第二步
这是比较简单的
直接去百度下载NET Framework 3.5
最好默认安装在C盘
安装完成以后 去控制面板启用功能或者你可以看到电脑已经帮你自动启用了（对应功能前面的方框为黑色就代表可启用了）
注意：NET Framework 3.5下面的两个子功能可以不启用

##第三步
如果上述步骤并不能解决你的问题
那么在控制面板启用NET Framework 3.5功能 最好window系统会出现报错的错误代码 一般有三个 用错误代码加上NET Framework 3.5百度搜索解决方法即可
